# Sistema Software FJ

Este proyecto corresponde al desarrollo de un sistema básico de gestión de clientes, servicios y reservas, implementado en Python utilizando programación orientada a objetos (POO).

---

## Descripción del proyecto

El sistema Software FJ permite simular el funcionamiento de una empresa que ofrece diferentes servicios como reservas de salas, alquiler de equipos y asesorías.  

A través del desarrollo de este proyecto se aplican conceptos fundamentales de programación, permitiendo gestionar información sin uso de bases de datos, únicamente mediante objetos y estructuras en memoria.

---

## Objetivo

Desarrollar un sistema funcional que integre:

- Registro de clientes
- Creación de servicios
- Gestión de reservas
- Manejo de errores mediante excepciones
- Registro de eventos en archivos de log
